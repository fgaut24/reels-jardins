Générateur de Reels — Les Jardins en scène

Déposez la configuration des soirées, lancez la génération, récupérez les vidéos.

1 · Connexion GitHub

Le jeton reste uniquement dans votre navigateur (stockage local). Utilisez un jeton « fine-grained » limité à ce seul dépôt, avec les permissions Contents : lecture/écriture et Actions : lecture/écriture. Ne partagez pas cette page avec le jeton enregistré.
Comment créer le jeton (pas à pas)
  1. GitHub → votre photo → Settings.
  2. Developer settings → Personal access tokens → Fine-grained tokens → Generate new token.
  3. Repository access → Only select repositories → choisissez votre dépôt.
  4. Repository permissions : Contents = Read and write ; Actions = Read and write (Metadata se met seul).
  5. Generate token, copiez-le, collez-le ci-dessus.

2 · Configuration des soirées

Déposez un fichier soirees.json ou modifiez directement ci-dessous.

Astuce : pour prévisualiser l'affiche animée avant de générer, utilisez l'application « Générateur de Reels » fournie séparément.
